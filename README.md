# Browser Webpages Project
This project presents web pages displayed in a browser. 
The purpose of this repository is purely personal. 
Therefore, it should not be evaluated in terms of aesthetics or visual design.

## Purpose
The main goal of this project is to experiment with basic web structures
and understand how web pages are rendered in a browser environment.

## Technologies Used
- HTML
- CSS
- JavaScript

## Notes
This repository is intended for educational and personal practice purposes.
